## 配置用户名密码

git config --global user.name  "username"  
git config --global user.email  "email"


## 初次提交
    git init
    git add README.md
    git commit -m "first commit"
    git remote add origin https://github.com/Chorlotte/test.git
    git push -u origin master
                
## 再次上传新文件

    git push -u origin master
